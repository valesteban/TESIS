# Libreria NetworkX
La biblioteca NetworkX es una biblioteca de Python para el estudio de grafos y análisis de redes. Proporciona una serie de funciones y métodos para crear, manipular y analizar grafos, así como para realizar cálculos y análisis sobre ellos.

Las principales características de NetworkX son las siguientes:

Soporte para diferentes tipos de grafos: NetworkX puede representar diferentes tipos de grafos, incluidos grafos simples, grafos dirigidos, grafos multidirigidos, grafos ponderados y grafos con atributos.
Conjunto completo de funciones y métodos: NetworkX proporciona un conjunto completo de funciones y métodos para crear, manipular y analizar grafos. Estas funciones y métodos incluyen operaciones básicas como la adición y eliminación de nodos y aristas, así como operaciones más complejas como el cálculo de rutas, componentes conectados y comunidades.
Cálculos y análisis avanzados: NetworkX también proporciona funciones y métodos para realizar cálculos y análisis avanzados sobre grafos. Estos cálculos y análisis incluyen el cálculo de medidas de centralidad, el análisis de conectividad y la detección de patrones.
